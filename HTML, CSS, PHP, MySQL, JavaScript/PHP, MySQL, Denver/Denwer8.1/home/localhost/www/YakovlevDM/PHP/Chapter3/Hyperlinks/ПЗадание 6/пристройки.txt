Оборонительное сооружение — постройка (сооружение, строение), предназначенная для укрытого размещения и наиболее эффективного применения вооружения, военной техники, пунктов управления, а также для защиты войск (сил), населения и объектов тыла государства от воздействия средств поражения противника.
